# Run hunt repeat

#### Game made in 48h for the Ludum Dare 51 with the help of @IntQuant

___

Every 10 seconds, the roles will change, you will either have to run, or hunt the enemies !

The goal is to survive as long as possible

## Controls

- WASD to move
